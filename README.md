# Stock Control System

Sistema de controle de estoque desenvolvido para fins de estudo utilizando Java, Spring Boot, Angular e PostgreSQL.

## Tecnologias

- Java 21
- Spring Boot
- Angular
- PostgreSQL
- Maven
- Git

## Objetivo

Desenvolver um sistema completo de controle de estoque seguindo práticas utilizadas em projetos profissionais.

## Funcionalidades previstas

- Cadastro de Categorias
- Cadastro de Fornecedores
- Cadastro de Produtos
- Entrada de Estoque
- Saída de Estoque
- Dashboard
- Login de Usuários
- Relatórios
